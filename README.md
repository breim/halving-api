## Requirements

For development, you will only need Node.js and a node global package, NPM, installed in your environement.

## Install

    $ git clone https://github.com/rafaelszago/express-js-api
    $ cd express-js-api
    $ npm install

## Development

    $ npm run dev

## Production

    $ npm run prod
